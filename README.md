# learn_thai

A Flutter application for learning Thai language with Shan script support. This app provides an interactive way to learn Thai consonants, vowels, tone rules, and common words with both English and Shan language interfaces.

## Features

- **Thai Language Learning**: Interactive modules for learning Thai consonants, vowels, and tone rules
- **Bilingual Interface**: User interface available in both English and Shan languages
- **Word Dictionary**: Comprehensive dictionary of common Thai-Shan word pairs
- **Sentence Builder**: Interactive tool for constructing Thai sentences
- **Syllable Lab**: Practice Thai syllable pronunciation and structure
- **Converter**: Convert between Thai and Shan scripts
- **Daily Word**: Learn a new word every day with audio pronunciation

## Project Structure

- `lib/`
  - `data/` - Application data and word lists
  - `models/` - Data models for words and UI components
  - `screens/` - Different learning modules and UI screens
  - `widgets/` - Reusable UI components
  - `main.dart` - Application entry point with theme configuration
